# 튜토리얼 E2E 플레이스루 체크리스트

## Chapter 0 A/B 개정 구간

1. 새 게임에서 작전 회의 대사에 에온의 사과, 아르온의 패스키 안내, 테우스의 왼쪽 안내가 모두 표시된다.
2. 테우스 소개 카드는 대화 뒤에 열리고 3초 동안 닫히지 않으며, 이후 키보드·마우스·게임패드 입력으로 세로 압축되어 닫힌다.
3. 오른쪽으로 잘못 이동하는 동작을 네 번 반복하면 일반 3회 반응과 붉은 자폭 장난 반응이 순서대로 나온다.
4. 왼쪽 끝에서 페이드 후 `Z01B_HiddenGlideRoom`으로 이동하고 카메라가 방 범위에 고정된다.
5. 활공 안내 패널이 대화와 겹치지 않고, 낙하 시 상승기류로 복구되며 패스키까지 이동할 수 있다.
6. 패스키 접촉 시 획득 상태가 저장되고 목표 화살표가 복귀 방향으로 바뀐다.
7. 회의장 복귀 후에만 사다리 출구가 열리며 기존 훈련·장비·외곽 전투 2개·헬테 흐름이 유지된다.
8. `TUTO_A_01`, `TUTO_B_01`, `TUTO_A_RETURN` 저장 상태에서 재실행해 각각 숨겨진 방 탐색, 숨겨진 방 입구, 회의장 복귀 지점이 복원되는지 확인한다.

## 시작 전

- `Narthex/Validation/Validate Active Tutorial Scene`가 통과한다.
- 개발 반복 테스트 중에는 `DevelopmentProgressResetManager`의 초기화 옵션이 켜져 있다.
- 최종 저장 동작 확인 시에는 해당 초기화 옵션을 끈다.

## 플레이 흐름

1. **아다마스 HQ**
   - 대화가 `Space`로 진행되고 대화 종료 후 소개 카드가 표시된다.
   - 마지막 대사에 `SPACE: 닫기`가 표시된다.
   - `A/D` 이동 목표와 비콘이 표시된다.
2. **훈련 구역**
   - `Left Shift` 대시, `Space` 점프·활공, 마우스 왼쪽 버튼 또는 `Enter` 공격, `2` 펄스가 현재 퀘스트 순서대로 목표를 완료한다.
   - 각 목표 변경 시 상단 목표 카드와 스테이지 캡션이 갱신된다.
3. **모듈 제어실**
   - `I`로 모듈 트리를 열고 닫을 수 있다.
   - `Tab`으로 인벤토리를 열고 닫을 수 있다.
4. **외곽 접근로**
   - 크리온 소개 카드가 표시된다.
   - 부츠 범위에서 `[ F ] 크리온 부츠 습득` 안내가 표시된다.
   - 더블 점프 획득 뒤 릴레이 범위에서 `[ F ] 릴레이 활성화` 안내가 표시된다.
5. **광물 저장고**
   - 헬테 소개 카드와 보스 전투가 정상 시작된다.
   - 헬테 처치 후 결과 화면, 완료 메시지, 챕터 1 이동 버튼이 표시된다.
   - 버튼 클릭과 `F` 모두 챕터 1 진입을 시도한다.

## UI / 오디오 확인

- 목표 카드, 대화창, 상호작용 프롬프트, 체력 UI가 서로 겹치지 않는다.
- SFX 클립 연결 뒤 내러티브 시작, 목표 갱신, 완료 효과음이 재생된다.
- `SfxVolume` 변경 시 이후 오디오 연결 작업에서 UI/월드 소스가 같은 설정을 사용하도록 확인한다.
