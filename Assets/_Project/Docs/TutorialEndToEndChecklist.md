# 튜토리얼 E2E 플레이스루 체크리스트

## 시작 전

- `Narthex/Validation/Validate Active Tutorial Scene`가 통과한다.
- 개발 반복 테스트 중에는 `DevelopmentProgressResetManager`의 초기화 옵션이 켜져 있다.
- 최종 저장 동작 확인 시에는 해당 초기화 옵션을 끈다.

## 플레이 흐름

1. **아다마스 HQ**
   - 소개 카드와 대화가 `F`로 진행된다.
   - 마지막 대사에 `F: 닫기`가 표시된다.
   - `A/D` 이동 목표와 비콘이 표시된다.
2. **훈련 구역**
   - `Space` 점프·활공, `Enter` 공격, `Left Shift` 대시, `2` 펄스가 순서대로 목표를 완료한다.
   - 각 목표 변경 시 상단 목표 카드와 스테이지 캡션이 갱신된다.
3. **모듈 제어실**
   - `I`로 모듈 트리를 열고 닫을 수 있다.
   - `Tab`으로 인벤토리를 열고 닫을 수 있다.
4. **외곽 접근로**
   - 크리온 소개 카드가 표시된다.
   - 부츠 범위에서 `[ F ] 크리온 부츠 습득` 안내가 표시된다.
   - 더블 점프 획득 뒤 릴레이 범위에서 `[ F ] 릴레이 활성화` 안내가 표시된다.
5. **광물 저장고**
   - 헬테 소개 카드와 보스 전투가 정상 시작된다.
   - 헬테 처치 후 결과 화면, 완료 메시지, 챕터 1 이동 버튼이 표시된다.
   - 버튼 클릭과 `F` 모두 챕터 1 진입을 시도한다.

## UI / 오디오 확인

- 목표 카드, 대화창, 상호작용 프롬프트, 체력 UI가 서로 겹치지 않는다.
- SFX 클립 연결 뒤 내러티브 시작, 목표 갱신, 완료 효과음이 재생된다.
- `SfxVolume` 변경 시 이후 오디오 연결 작업에서 UI/월드 소스가 같은 설정을 사용하도록 확인한다.

## 완료 기준

- 새 저장에서 튜토리얼을 처음부터 끝까지 막힘 없이 완료한다.
- 결과 화면에서 챕터 1로 넘어간다.
- 재시작 후 저장 정책(개발 초기화 켜기/끄기)이 의도대로 동작한다.
