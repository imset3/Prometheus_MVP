# 21. 사운드 시스템

## 1. 사운드 시스템 정의

사운드 시스템은 튜토리얼과 CHAPTER_01의 진행 상황, 전투, UI 조작, 보스전, 회차 재시작에 맞춰 BGM과 SFX를 재생하는 시스템이다.

사운드는 BGM과 SFX로 분류한다.

사운드 볼륨은 설정 시스템의 전체 볼륨, BGM 볼륨, 효과음 볼륨, UI 사운드 볼륨 값을 따른다.

## 2. BGM 기본 규칙

| ID | 규칙 | 설명 | 우선순위 |
| --- | --- | --- | --- |
| BGM-RULE-01 | BGM은 구간별로 분리한다. | 튜토리얼, CHAPTER_01, 보스전, 결과 화면에 맞춰 전환한다. | 상 |
| BGM-RULE-02 | CHAPTER_01 범위만 우선 정의한다. | 현재 구현 범위 밖의 챕터 BGM은 작성하지 않는다. | 상 |
