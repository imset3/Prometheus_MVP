# 05. 스테이터스

## 1. 스테이터스 정의

스테이터스는 플레이어와 전투 대상의 수치 상태를 나타낸다.

레벨과 경험치는 회차 데이터이며 회차 재시작 시 초기화된다.

모듈 패시브와 강화 효과는 현재 회차 안에서만 스테이터스에 영향을 준다.

## 4. 회차 초기화 기준

| 항목 | 처리 |
| --- | --- |
| 현재 HP | 회차 시작 시 기본값으로 초기화 |
