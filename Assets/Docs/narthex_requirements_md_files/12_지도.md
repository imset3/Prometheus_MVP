# 12. 지도

## 1. 지도 정의

지도는 현재 회차에서 플레이어의 위치, 스테이지, 목표, 송신탑, 포탈 정보를 표시하는 UI이다.

지도 탐색 상태는 CHAPTER_01 구현 범위에서는 회차 데이터로 취급하고, 회차 재시작 시 초기화한다.

## 2. 지도 요구사항

| ID | 기능 | 요구사항 | 우선순위 |
| --- | --- | --- | --- |
| MAP-REQ-01 | 지도 열기 | Tab 키를 눌러 지도 UI를 열 수 있어야 한다. | 상 |
| MAP-REQ-02 | 지도 닫기 | Tab 또는 Esc로 지도 UI를 닫을 수 있어야 한다. | 상 |
| MAP-REQ-03 | 현재 위치 표시 | 현재 스테이지 내 플레이어 위치를 표시한다. | 중 |
| MAP-REQ-04 | 현재 스테이지 표시 | 현재 STG ID와 구역 정보를 표시한다. | 중 |
| MAP-REQ-05 | 목표 위치 표시 | 추적 중인 퀘스트 목표 위치를 표시한다. | 중 |
| MAP-REQ-06 | 송신탑 표시 | 현재 회차의 송신탑 상태를 표시한다. | 중 |
| MAP-REQ-07 | 포탈 표시 | 포탈 활성화 여부와 위치를 표시한다. | 중 |
| MAP-REQ-08 | 직접 이동 제한 | 지도 안에서 순간이동하거나 스테이지 이동하지 않는다. | 상 |
| MAP-REQ-09 | 회차 초기화 | 회차 재시작 시 지도 상태를 STG-001 기준으로 갱신한다. | 상 |
