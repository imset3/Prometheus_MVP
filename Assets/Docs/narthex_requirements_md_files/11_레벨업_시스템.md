# 11. 레벨업 시스템

## 1. 레벨업 시스템 정의

레벨업 시스템은 현재 회차 안에서만 동작하는 성장 시스템이다.

플레이어는 ENEMY 처치, 전투 구역 클리어, 스테이지 클리어, 보스 처치로 경험치를 획득한다.

필요 경험치에 도달하면 레벨이 상승하고, 레벨 보상으로 모듈 포인트 또는 회차성 보상을 획득한다.

레벨, 경험치, 레벨 보상 지급 상태는 회차 데이터이며 사망 또는 회차 재시작 시 초기화한다.

## 2. 레벨업 기본 규칙

| ID | 규칙 | 설명 | 우선순위 |
| --- | --- | --- | --- |
| LV-RULE-01 | 플레이어는 매 회차 Lv.1부터 시작한다. | 회차 재시작 시 현재 레벨은 Lv.1로 초기화한다. | 상 |
| LV-RULE-02 | CHAPTER_01 기준 최대 레벨은 Lv.10이다. | 현재 범위는 튜토리얼과 CHAPTER_01이므로 Lv.10까지 우선 정의한다. | 상 |
| LV-RULE-03 | 경험치는 회차 데이터이다. | 사망 시 현재 경험치와 누적 경험치를 초기화한다. | 상 |
| LV-RULE-04 | 초과 경험치는 이월된다. | 한 번의 경험치 획득으로 여러 레벨 상승이 가능하다. | 상 |
| LV-RULE-05 | 레벨업 보상은 회차성 보상이다. | 모듈 포인트와 소모품은 회차 재시작 시 초기화한다. | 상 |
| LV-RULE-06 | 보스 모듈 트리 해금은 레벨 보상이 아니다. | 보스 처치 보상은 영구 데이터로 별도 처리한다. | 상 |
