# 01. 게임 개요

## 1. 게임 개요

| 항목 | 내용 |
| --- | --- |
| 게임 제목 | 나르텍스 (Narthex) |
| 장르 | 2D 액션 플랫포머 / 로그라이크식 회차 진행 |
| 핵심 테마 | 스팀펑크, 기술 독점, 금지 기술 탈취, 혁명 |
| 플레이 방식 | 튜토리얼 이후 CHAPTER_01 스테이지 순차 진행 |
| 플레이 인원 | 1인 |
| 개발 엔진 | Unity |
| 우선 플랫폼 | PC Windows |
| 현재 목표 | 졸업작품 전시회용 튜토리얼 + CHAPTER_01 완성 |
| 핵심 경험 | 플레이어는 기본 모듈 트리와 보스 처치로 얻은 보스 모듈 트리를 회차마다 다시 구성하며 지배자의 기술로 지배자를 무너뜨린다. |

## 2. 핵심 진행 흐름

```txt
프롤로그
→ 튜토리얼
→ 튜토리얼 보스
→ CHAPTER_01 STG-001
→ CHAPTER_01 일반 스테이지 진행
→ CHAPTER_01 중간 보스
→ CHAPTER_01 최종 보스
→ 결과 화면
```

## 3. 로그라이크 회차 구조

| ID | 규칙 | 설명 | 우선순위 |
| --- | --- | --- | --- |
| GAME-LOOP-01 | 사망 시 회차 재시작 | 플레이어가 사망하면 현재 회차 진행을 종료하고 CHAPTER_01 STG-001부터 다시 시작한다. | 상 |
| GAME-LOOP-02 | 회차 데이터 초기화 | 레벨, 경험치, 모듈 포인트, 모듈 해금/강화/장착 상태, 스테이지 진행, 송신탑 상태는 초기화한다. | 상 |
| GAME-LOOP-03 | 영구 해금 유지 | 기본 모듈 트리 접근 권한과 보스 처치로 얻은 보스 모듈 트리는 사망 후에도 유지한다. | 상 |
| GAME-LOOP-04 | 모듈 포인트 회차 자원 | 모듈 포인트는 회차 중 획득하고 회차 종료 시 초기화한다. | 상 |

## 6. 튜토리얼 규칙

| ID | 규칙 | 설명 | 우선순위 |
| --- | --- | --- | --- |
| TUTO-RULE-01 | 튜토리얼은 헬테 처치를 목표로 한다. | 기본 조작과 전투 학습 뒤 튜토리얼 보스를 처치하면 CHAPTER_01 진행으로 넘어간다. | 상 |
| TUTO-RULE-02 | 기본 모듈 트리를 학습한다. | 튜토리얼에서는 모듈 트리 접근, 모듈 해금, 모듈 장착, 모듈 사용 흐름을 안내한다. | 상 |
| TUTO-RULE-03 | 튜토리얼 난이도는 낮게 설정한다. | 사망 가능성은 낮추되, 회차 재시작 구조는 설명한다. | 중 |
| TUTO-RULE-04 | 보스 패턴 상세는 별도 문서에서 보강한다. | 현재 문서에서는 보스전 진입/처치/보상 연결만 정의한다. | 상 |
