# 04. 송신탑 규칙

## 1. 송신탑 정의

송신탑은 현재 회차 안에서 ENEMY 버프를 해제하고 진행 상태를 저장하는 보조 장치이다.

송신탑은 부활 지점이 아니다. 플레이어가 사망하면 송신탑 위치와 관계없이 CHAPTER_01 STG-001 시작 지점에서 새 회차를 시작한다.

## 2. 기본 구조

| 구분 | 역할 |
| --- | --- |
| 상태 | CHECK_POINT_ENEMY / CHECK_POINT_PLAYER |
| 상태 전환 | ENEMY에서 PLAYER로만 전환 |
| ENEMY 상태 | 해당 구역 ENEMY 버프 적용 |
| PLAYER 상태 | ENEMY 버프 해제, 회차 진행 저장 |
| 상호작용 | F 입력으로 활성화 |
| 회차 재시작 | 모든 송신탑 상태 초기화 |

## 3. 송신탑 요구사항

| ID | 분류 | 기능 | 조건 | 결과 | 우선순위 |
| --- | --- | --- | --- | --- | --- |
| CHK-01 | 상태 | 송신탑은 ENEMY 또는 PLAYER 상태 중 하나만 가진다. | 스테이지 로드 시 | 현재 회차 데이터 기준 상태 적용 | 상 |
| CHK-02 | ENEMY 버프 | ENEMY 상태일 때 해당 구역 ENEMY에게 버프를 제공한다. | 스테이지 입장 | ENEMY 버프 적용 | 상 |
| CHK-03 | 상호작용 | 플레이어가 F키로 송신탑을 활성화할 수 있다. | ENEMY 상태 + 상호작용 거리 | PLAYER 상태로 전환 | 상 |
| CHK-04 | 전환 제한 | PLAYER 상태는 ENEMY 상태로 되돌아가지 않는다. | 같은 회차 안 | 역전환 없음 | 상 |
| CHK-05 | 저장 | PLAYER 상태 전환 순간 회차 진행 상태를 저장한다. | 활성화 성공 | 회차 데이터 저장 | 상 |
| CHK-06 | 부활 제외 | 송신탑은 사망 후 부활 위치로 사용하지 않는다. | 플레이어 사망 | STG-001 시작 지점으로 회차 재시작 | 상 |
| CHK-07 | 회차 초기화 | 회차 재시작 시 송신탑 상태는 초기화된다. | 회차 재시작 | 모든 송신탑 ENEMY 상태 | 상 |

| BUF-01 | 공격력 증가 | 공격 보조 | ENEMY | 공격력 +10% | 상 |
| BUF-02 | 공격 속도 증가 | 공격 보조 | ENEMY | 공격 속도 +10% | 상 |
| BUF-03 | 방어막 | 생존 보조 | ENEMY | 최대 체력의 10% 보호막 | 상 |
| BUF-04 | 이동 속도 증가 | 이동 보조 | ENEMY | 이동 속도 +10% | 중 |
