# 13. 무기

## 1. 무기 정의

현재 무기는 총검 한 종류로 고정한다.

일반 공격은 총검의 검 부분을 사용하는 근거리 공격이다.

원거리 모듈 사용 시 총 모션으로 전환할 수 있지만, 모듈 발동 위치와 연출은 데우스가 담당한다.

## 2. 무기 구조

| 항목 | 내용 | 우선순위 |
| --- | --- | --- |
| 무기 종류 | 총검 | 상 |
| 기본 공격 방식 | 근거리 검 공격 | 상 |
| 원거리 표현 | 원거리 모듈 사용 시 총 모션 보조 | 상 |
| 모듈 발동 | 데우스 발동 연출과 함께 처리 | 상 |
| 무기 수치 | 현재 단계에서는 별도 정의하지 않음 | 중 |

## 3. 공격 방식 구분

| 공격 구분 | 사용 무기 / 오브젝트 | 설명 | 관련 시스템 |
| --- | --- | --- | --- |
| 일반 공격 | 검 | 총검의 검 부분으로 근거리 공격 | 입력 / 전투 |
| 근거리 모듈 | 데우스 + 검 보조 모션 | 데우스 기준 근접 판정 | 모듈 / 전투 |
| 원거리 모듈 | 데우스 + 총 보조 모션 | 데우스 기준 투사체 발사 | 모듈 / 전투 |
| 이동 모듈 | 데우스 | 이동, 공중 제어, 위치 변경 | 모듈 / 이동 |

## 4. 무기 모션 규칙

| ID | 규칙 | 설명 | 우선순위 |
| --- | --- | --- | --- |
| WPN-MOTION-01 | 기본 상태는 검 사용 모션이다. | 일반 공격 입력 시 검 모션을 출력한다. | 상 |
| WPN-MOTION-02 | 원거리 모듈 사용 시 총 모션을 보조로 사용할 수 있다. | 시각적 전환이며 수치 변경은 아니다. | 상 |
| WPN-MOTION-03 | 모션 전환은 무기 교체가 아니다. | 같은 총검 안에서 사용 방식만 달라진다. | 상 |
| WPN-MOTION-04 | 모듈 판정은 전투 시스템이 처리한다. | 무기 문서는 판정 수치를 확정하지 않는다. | 상 |
