# 19. 설정 시스템

## 1. 설정 시스템 정의

설정 시스템은 플레이어가 게임 플레이 환경을 조정할 수 있도록 제공되는 옵션 기능이다.

설정 시스템은 회차, 모듈 트리, 전투, 성장 데이터에 직접적인 영향을 주지 않는다.

설정 데이터는 회차 재시작과 관계없이 유지된다.

## 2. 설정 분류

| 설정 분류 | 포함 항목 | 우선순위 |
| --- | --- | --- |
| 사운드 설정 | 전체 볼륨, BGM, 효과음, UI 사운드 | 상 |
| 화면 설정 | 해상도, 전체화면, 창모드, 화면 밝기 | 상 |
| 조작 설정 | 키 설정 확인, 키 변경 | 중 |
| 게임 플레이 보조 설정 | 화면 흔들림, 피해 숫자, 상태이상 표시, 목표 안내 | 중 |
