# 03. 이동 시스템

## 1. 이동 분류 조건

이동은 플레이어의 위치, 속도, 중력, 이동 상태를 바꾸는 기능이다.

공격 판정, 피해 처리, 모듈 해금, 보스 모듈 트리 해금은 이동 기능으로 분류하지 않는다.

## 2. 이동 요구사항

| ID | 기능 | 참고 기본 키 | 요구사항 |
| --- | --- | --- | --- |
| MOV-01 | 좌우 이동 | A / D | 지상 및 공중에서 수평 이동할 수 있어야 한다. |
| MOV-02 | 위 입력 | W | 위쪽 보기 또는 위 방향 조준 요청을 제공한다. |
| MOV-03 | 아래 입력 | S | 아래쪽 보기 또는 아래 방향 조준 요청을 제공한다. |
| MOV-04 | 점프 | Space | 지상에서 점프할 수 있어야 한다. |
| MOV-05 | 더블 점프 | Space | 공중에서 추가 점프를 1회 사용할 수 있어야 한다. |
| MOV-06 | 활공 | Space 유지 | 공중에서 낙하 속도를 조절할 수 있어야 한다. |
| MOV-07 | 대시 | Shift | 입력 방향 또는 바라보는 방향으로 빠르게 이동할 수 있어야 한다. |

## 3. 활공 규칙

| ID | 항목 | 내용 |
| --- | --- | --- |
| GLD-01 | 발동 조건 | 점프 이후 공중 상태에서 Space 키를 누르고 있을 때 발동된다. |
| GLD-02 | 종료 조건 | Space를 떼거나 착지하거나 피격/특정 모듈 사용 시 종료된다. |
| GLD-03 | 전투 연동 | 활공 상태에서도 일반 공격과 모듈 사용이 가능하다. |
| GLD-04 | 목적 | 장거리 비행이 아니라 낙하 속도 조절과 착지 위치 조정에 사용한다. |

## 4. 대시 규칙

| ID | 항목 | 내용 |
| --- | --- | --- |
| DASH-01 | 발동 조건 | 대시 키 입력 시 발동된다. |
| DASH-02 | 방향 | 좌우 입력을 우선 적용하고, 입력이 없으면 바라보는 방향으로 대시한다. |
| DASH-03 | 공중 대시 | 공중에서도 사용할 수 있으며 착지 전 1회만 가능하다. |
| DASH-04 | 무적 연동 | 대시 중에만 피격 무적이 적용된다. |
| DASH-05 | 종료 | 지속 시간이 끝나면 일반 이동 상태로 복귀한다. |

## 5. 이동 임시 수치

| 항목 | 임시 값 | 설명 |
| --- | --- | --- |
| 이동 속도 | 7.0 | 좌우 기본 이동 속도 |
| 점프 힘 | 13.0 | 지상 점프 힘 |
| 더블 점프 힘 | 11.0 | 공중 추가 점프 힘 |
| 일반 중력 | 4.0 | 평상시 중력 값 |
| 일반 최대 낙하 속도 | -18.0 | 일반 낙하 시 최대 속도 |
| 활공 중력 | 1.0 | 활공 중 적용되는 중력 값 |
| 활공 낙하 속도 | -4.0 | 활공 중 최대 낙하 속도 |
| 대시 속도 | 18.0 | 대시 순간 이동 속도 |
| 대시 지속 시간 | 0.18초 | 대시 유지 시간 |
| 대시 쿨타임 | 0.5초 | 다음 대시까지 대기 시간 |
